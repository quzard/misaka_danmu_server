# 更新日志

## v2.7.6

### 新功能

- **两步验证与 PassKey** — 新增 TOTP 两步验证和 PassKey(WebAuthn) 登录体系，并补齐管理界面、注册流程与登录弹窗。
- **缓存管理页面** — 新增缓存统计、搜索、删除与键值预览能力，补齐前端弹窗管理入口。
- **顶部导航图标收纳** — 顶部导航接入 iconfont，支持图标/文字切换与挤压态展示。
- **液态玻璃页面样式** — 新增液态玻璃风格页面样式与底部悬浮导航胶囊。
- **流控面板 SSE 推送** — 流控状态面板与导航栏指示器改为 SSE 实时推送，减少轮询开销。

### 安全增强

- **PassKey HTTPS 限制** — PassKey 注册仅允许 HTTPS，避免同 IP 下与其他应用的 RP ID 冲突。

### 修复


- **登录与会话问题** — 修复 stale 登录 token、退出清理不可靠、`expiresIn=-1` 立即过期、登录跳转方式等问题。
- **PassKey / MFA 兼容性** — 修复反代 HTTPS 下 PassKey 注册 origin 不匹配、MFA 登录弹窗不弹出等问题。
- **PWA 缓存问题** — 修复 Service Worker 错误缓存 `index.html` 与 SPA fallback 缓存头问题。
- **容器与更新流程问题** — 修复更新流程中网络配置丢失、重启与更新判定相关问题。
- **暗色与卡片布局问题** — 修复登录页、登录卡片缓存按钮、日志预览卡片、令牌授权卡片等暗色适配与布局细节。

### 优化

- **RateLimitIndicator 视觉优化** — 流控指示器改为更柔和的主题色边框与背景，并同步适配液态玻璃风格。
- **液态玻璃样式兼容优化** — 修复输入框双层边框、Input.Search 圆角、Tooltip、Tag、Switch、Progress、Tabs 等组件在液态玻璃模式下的显示问题。
- **顶部导航与移动端体验优化** — 顶部导航图标与文字在不同宽度下自动切换，移动端与桌面端间距同步调整。
- **流控面板布局优化** — 调整流控面板在移动端与桌面端的卡片间距与整体层次感。
- **搜索源与页面间距优化** — 调整搜索源、资源仓库、弹幕搜索源等页面卡片间距，提升可读性。

### 维护

- **前端 / PWA 重构** — 抽离前端与 PWA 路由，修复 Service Worker 资源服务流程。
- **安全审计修复** — 完成一轮安全审计修复，覆盖认证、会话与登录相关问题。
- **文档同步** — 更新示例弹幕源与分集过滤相关说明，并持续同步变更记录。
